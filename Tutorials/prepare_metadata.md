## KEGG pathway and metabolite database preparation

This module provides scripts and precomputed files for preparing KEGG-based pathway and metabolite annotation resources.

The script `get_kegg_infos.R` retrieves human KEGG pathways, extracts pathway-associated compounds, collects compound exact mass information, and constructs reaction-based metabolite links. The generated files are used for m/z-based metabolite matching, pathway-level aggregation, and downstream metabolic network analysis.

Precomputed KEGG files are provided in this repository. Users can directly use these files for standard human metabolomics analysis. If another database or organism is required, users should prepare equivalent pathway-metabolite and metabolite-mass mapping files following the same format.

## Database preparation

MassLinker Suite provides precomputed KEGG-based pathway and metabolite annotation files for human metabolomics analysis. These files are generated by `R/get_kegg_infos.R`.

The script retrieves human KEGG pathways, extracts pathway-associated compounds, collects exact mass information for each compound, and constructs reaction-based metabolite links. These resources are used for m/z-based metabolite matching, pathway-level aggregation, and metabolic network analysis.

### Provided files

| File | Description |
|---|---|
| `compounds_detail_res.Rda` | Compound-level exact mass table with `id`, `exact_mass`, `mz_min`, and `mz_max` |
| `kegg_pathways.Rda` | List of KEGG pathways and their associated compounds |
| `kegg_pathways_id.Rda` | KEGG pathway identifiers retained for pathway analysis |
| `kegg_pathway_info.Rda` | Raw KEGG pathway-compound query results |
| `kegg_pathway_info_detail.Rda` | Intermediate KEGG annotation environment |
| `reactions.Rda` | KEGG reaction list |
| `formulars.Rda` | Parsed reaction equations, including reactants, products, and enzymes |
| `links.Rda` | Reaction-derived link table for metabolic network analysis |

### Using another database

Users can replace KEGG with another database, such as HMDB, Reactome, MetaCyc, LipidMaps, or a customized metabolite library. To use a custom database, please prepare equivalent files with the following information:

1. A compound mass table containing:
   - compound ID
   - exact mass
   - lower m/z boundary
   - upper m/z boundary

2. A pathway-compound mapping table containing:
   - pathway ID
   - pathway name
   - compound ID

3. A pathway information table containing:
   - pathway ID
   - pathway name
   - database source
   - organism, if applicable

4. Optional reaction or network links containing:
   - source node
   - target node
   - relationship type, if available

The most important file is the compound mass table, because it is used to match LC–MS features to candidate metabolites based on m/z tolerance. Pathway-compound mapping files are required for pathway-level aggregation and biological interpretation.
